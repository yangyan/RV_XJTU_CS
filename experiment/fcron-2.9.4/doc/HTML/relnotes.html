1.4. Release notes : fcron 2.9.4

  • PAM support has not been tested thoroughly. Help from a PAM expert would be appreciated (to audit the PAM code in fcron, and test it). You can disable PAM use in configure, using "--with-pam=no".

    Moreover, fcrondyn does not use PAM at this time (should it anyway ?).

  • Some commands of fcrondyn are not implemented yet.

  • Fcrondyn might be not very secure, and might be used to block fcron.
